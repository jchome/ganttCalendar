API - Méthodes d'instance

Vector(<début>, <longueur>)

Constructeur de l'objet Vector. Il prend 2 paramètres optionnels :
  • début : position du vecteur (position d'origine)
  • longueur : taille du vecteur
La position de fin est calculée à partir de ces 2 données.

Exemple: 
v1 = new Vector(4,10);

int getBegin()

Accès à la position d'origine du vecteur.

Exemple: 
v1.getBegin();
// retourne 4

int getEnd()

Accès à la position de fin du vecteur.

Exemple: 
v1.getEnd();
// retourne 14

int getLength()

Retourne la longueur du vecteur.

Exemple: 
v1.getEnd();
// retourne 10

Vector setBegin(<nouvelle position de début>)

Réaffecte la position de début du vecteur. Cette méthode retourne le vecteur.

Exemple: 
v1.setBegin(9);

Vector setEnd(<nouvelle position de fin>)

Réaffecte la position de fin du vecteur en changeant sa taille. Cette méthode retourne le vecteur.

Exemple: 
v1.setEnd(12);

Vector setLength(<nouvelle taille>)

Change la taille du vecteur. La posiution de fin est recalculée suite à cette opération. Cette méthode retourne le vecteur.

Exemple: 
v1.setLength(2);

boolean isNull()

Teste si le vecteur est nul (position et longueur nulles)

Exemple: 
v1.isNull();
// retourne false

string toString()

Retourne une chaine de caractère représentant les données du vecteur au format "V[DEBUT,FIN](L:LONGUEUR)".

Exemple: 
v1.toString();
// retourne "V[9,11](L:2)"

Vector union(<Vector>)

Exemple: 
var v1 = new Vector(4,10);

boolean containsValue<(<int>)

Retourne VRAI si le vecteur contient la valeur passée en paramètre.

Array allValues()

Retourne un tableau avec les valeurs qui sont contenues dans le vecteur.

JQueryObject niceDisplay( <containerObj>, <startRulerValue>, <endRulerValue>, <cssClass>)

Alimente l'objet containerObj d'une règle graduée et d'un trait pour représenter le vecteur.
  • containerObj : objet JQuery à alimenter
  • startRulerValue : valeur de début de la règle
  • endRulerValue : valeur de début de la règle
  • cssClass (optionnel) : classe CSS à ajouter sur les barres représentant les vecteurs


Exemple: 
var v1 = new Vector(4,2);
v1.niceDisplay($("#myDiv"),0,20);

arrayOfVector Vector.intersectAll(<arrayOfArrayOfVector>)

Réalise l'intersection entre chaque ligne de vecteurs. Les valeurs présentes dans toutes les lignes seront récupérées.

Exemple: 
var v1 = new Vector(3,7);
var v2 = new Vector(12,2);
var v3 = new Vector(6,4);
var v4 = new Vector(13,2);
var v5 = new Vector(2,5);
var v6 = new Vector(13,1);
var res = Vector.intersectAll( [[v1, v2], [v3, v4], [v5, v6]] );

JQueryObject Vector.niceDisplay(<arrayOfVector>, <containerObj>, <startRulerValue>, <endRulerValue>, <cssClass>)

Alimente l'objet containerObj d'une règle graduée et de traits pour représenter les vecteurs de la liste arrayOfVector.
  • arrayOfVector : tableau avec les vecteurs à afficher
  • containerObj : objet JQuery à alimenter
  • startRulerValue : valeur de début de la règle
  • endRulerValue : valeur de début de la règle
  • cssClass (optionnel) : classe CSS à ajouter sur les barres représentant les vecteurs


Exemple: 
Vector.niceDisplay([new Vector(4,2) , new Vector(8,7)], $("#vect_niceDisplay_1"),0,20);
